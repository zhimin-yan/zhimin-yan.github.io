Web Crawler

The Simple Webcrawler App takes an array of URLs and depths to crawl the web, building a graph of the websites. Using the Breadth-First Search (BFS) algorithm, it iteratively traverses URLs, avoids duplicates, and skips failed fetches. It also prints the execution time and optimizes performance with limited concurrent requests to enhance speed and reliability.
